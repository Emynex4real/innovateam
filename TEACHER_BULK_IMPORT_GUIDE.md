# 📚 Teacher's Guide: Bulk Question Import

## 🚀 Quick Start (3 Steps)

### **Step 1: Prepare Your Questions**
Copy your questions from anywhere (Word, PDF, notes, etc.)

### **Step 2: Paste & Parse**
1. Go to **Tutor → Questions → 📋 Bulk Import**
2. Fill in: Subject, Topic, Category, Difficulty
3. Paste your questions
4. Click **"🚀 Parse with AI"**

### **Step 3: Review & Save**
1. Check all parsed questions
2. Edit if needed
3. Click **"Save All Questions"**

**Done!** ✅

---

## 📝 Supported Formats

### **Format 1: Standard Numbered**
```
1. What is 2 + 2?
A. 3
B. 4
C. 5
D. 6
Answer: B
Explanation: 2 + 2 equals 4
```

### **Format 2: With Parentheses**
```
1) What is the capital of Nigeria?
A) Lagos
B) Abuja
C) Kano
D) Port Harcourt
Correct: B
```

### **Format 3: Q Prefix**
```
Q1. Solve: x + 5 = 10
A. x = 3
B. x = 5
C. x = 10
D. x = 15
Answer: B
Explanation: Subtract 5 from both sides
```

### **Format 4: Minimal**
```
What is photosynthesis?
A. Plant respiration
B. Plant food production
C. Plant reproduction
D. Plant growth
Answer: B
```

## ✅ Tips for Best Results

### **DO:**
- ✅ Include all 4 options (A, B, C, D)
- ✅ Clearly mark the correct answer
- ✅ Add explanations (or AI will generate)
- ✅ Use consistent formatting
- ✅ Separate questions with blank lines
- ✅ Start with 10-20 questions to test

### **DON'T:**
- ❌ Mix different numbering styles
- ❌ Skip options (must have A, B, C, D)
- ❌ Forget to mark correct answers
- ❌ Use special characters excessively
- ❌ Paste more than 50 questions at once

## 🔧 Troubleshooting

### **Problem: AI didn't parse correctly**
**Solution:** 
- Check your format matches examples
- Ensure all questions have 4 options
- Make sure answers are clearly marked
- Try smaller batches

### **Problem: Missing explanations**
**Solution:**
- AI will generate basic explanations
- You can edit them after parsing
- Add your own explanations in the text

### **Problem: Wrong answer detected**
**Solution:**
- Review parsed questions carefully
- Edit the correct answer in the review screen
- Make sure "Answer:" is clearly marked in text

### **Problem: Some questions skipped**
**Solution:**
- Check if they have all 4 options
- Ensure consistent formatting
- Look for special characters
- Try parsing those questions separately

## 💡 Pro Tips

### **Tip 1: Use Templates**
Create a template in Word/Notes:
```
1. [Question]
A. [Option A]
B. [Option B]
C. [Option C]
D. [Option D]
Answer: [Letter]
Explanation: [Why]
```

### **Tip 2: Batch by Topic**
Import questions in batches by topic:
- Monday: Algebra questions
- Tuesday: Geometry questions
- Wednesday: Word problems

### **Tip 3: Review Before Saving**
Always review AI-parsed questions:
- Check mathematical notation
- Verify correct answers
- Improve explanations
- Fix any formatting issues

### **Tip 4: Save Time on Explanations**
Let AI generate basic explanations, then:
- Edit to add more detail
- Include formulas
- Add teaching tips
- Reference textbook pages

# Math Equation Guide for JAMB Students 📐

## How to Write Math Equations in the Forum

### Basic Syntax
Wrap your equations in dollar signs:
- **Inline math:** `$equation$` - appears in the same line as text
- **Block math:** `$$equation$$` - appears centered on its own line

## Tips for Writing Clear Math

### 1. Use Block Math for Important Equations
```
For the main formula, use block math:

$$E = mc^2$$

This makes it stand out!
```

### 2. Explain Your Variables
```
Given $v = u + at$ where:
- $v$ = final velocity
- $u$ = initial velocity
- $a$ = acceleration
- $t$ = time
```

### 3. Show Your Work Step-by-Step
```
Solve $2x + 5 = 15$

Step 1: $2x = 15 - 5$
Step 2: $2x = 10$
Step 3: $x = \frac{10}{2}$
Step 4: $x = 5$
```

### 4. Use Parentheses for Clarity
```
Bad: $1/2x$ (ambiguous)
Good: $\frac{1}{2}x$ or $\frac{1}{2x}$
```

## Common Mistakes to Avoid

### ❌ Forgetting Dollar Signs
```
Wrong: x^2 + y^2 = z^2
Right: $x^2 + y^2 = z^2$
```

### ❌ Unmatched Braces
```
Wrong: $\frac{1{2}$
Right: $\frac{1}{2}$
```

### ❌ Missing Backslashes
```
Wrong: $sqrt{16}$
Right: $\sqrt{16}$
```
